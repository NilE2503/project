# Утилита для сортировки данных.
Утилита запускается из командной строки.

## Источники данные:
* Текстовый файл.
* Ввод пользователя с консоли.

##Методы сортировки:
*Пузырьком.
*Вставки.
*Выбором.

## Входные данные.
На вход программы поступают целые числа.

## Сохранение данных
Сохраненные данные сохраняются в файл out.txt
